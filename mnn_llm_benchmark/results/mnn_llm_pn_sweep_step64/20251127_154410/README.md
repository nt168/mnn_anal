# mnn_llm_pn_sweep_step64

## 任务描述

单线程（t=1）下的大范围 p/n 组合测试。 固定参数：rep=3, c=2, mmp=0, dyo=0, vp=1, kv=true。 p, n ∈ [32..1024]，步进为 64。 由于从 32 开始以 64 为步长，实际可达最大值为 992（无法整步到 1024）。


## 执行配置

| 配置项 | 值 |
|--------|-----|
| 超时时间 | 600秒 |
| 使用模型 | qwen2_5_0_5b, llama_3_2_1b, smolvlm2_256m |

## 基准套件

### pn_sweep_step64

**描述**: p, n 从 32 开始到约 1024，步长为 64 的笛卡尔积扫描。 实际取值为 [32, 96, ..., 992]（共 16 个点），p 与 n 全组合，共 256 组。


**用例数量**: 768

**变量定义**:
- n_prompt: 32 到 1024 (步长: 64)
- n_gen: 32 到 1024 (步长: 64)
