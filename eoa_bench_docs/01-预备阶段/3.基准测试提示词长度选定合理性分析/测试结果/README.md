# 测试结果说明

本目录用于存储提示词长度测试的输出结果和分析报告。

## 结果文件
- **length_X_Y.json**: 提示词长度X第Y次测试的详细结果
- **stability_analysis.json**: 汇总分析结果和稳定性结论

## 分析内容
- **稳定性分析**: 不同长度下Decode阶段的变异系数
- **效率分析**: 测试时间随长度的增长趋势
- **综合评估**: 稳定性-效率平衡的最优长度区间

## 预期输出
测试完成后，将提供：
1. 各长度的稳定性指标排序
2. 效率随长度变化的趋势图
3. 推荐的提示词长度范围
4. 对1号报告的建议
