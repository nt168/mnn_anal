# 测试脚本说明

本目录包含提示词长度测试的执行脚本和配置文件。

## 使用方法
执行完整的提示词长度分析测试：
```bash
./run_prompt_length_test.sh
```

## 配置文件
- **prompt_length_test.yaml**: MNN_LLM_Benchmark任务配置，包含12个提示词长度测试套件

## 测试流程
1. 检查环境和模型配置
2. 按长度梯度执行测试（8→384）
3. 每个长度重复10次
4. 收集稳定性和效率数据
5. 生成汇总分析结果

## 输出结果
- **length_X_Y.json**: 每个长度每次测试的详细结果
- **stability_analysis.json**: 汇总分析结果

---

**注意**: 测试可能需要较长时间，建议在稳定环境中执行。
