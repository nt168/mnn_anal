# 3.基准测试提示词长度选定合理性分析

## 文档概述
本文档通过系统性实验测试分析不同提示词长度对端侧LLM推理性能的影响，选定既能给出稳定结果又不太消耗时间的长度区间，为后续测试提供长度约束。

## 快速开始
1. 阅读主分析文档了解实验设计
2. 运行长度测试脚本（12个长度×10次重复）
3. 分析稳定性和效率数据，选定最优区间
4. 为4号报告提供长度约束建议

## 实验状态
- **状态**：脚本已完成，可直接执行
- **测试长度**：8-384，共12个梯度点
- **重复次数**：每个长度10次，共120组测试

## 主要内容
- **分析对象**：不同长度提示词对llm_bench输出指标的稳定性影响
- **分析方法**：方差分析+时间效率评估（实验验证）
- **分析过程**：8-384长度梯度，每个长度10次重复测试
- **分析结论**：稳定性-效率平衡的最优长度区间选定
- **输入依赖**：基于1号报告模型和2号报告指标合理性基础
- **输出约束**：为4号报告的固定模式验证提供长度约束

## 核心问题
- **短提示词问题**：Decode结果波动大，方差不稳定
- **长提示词问题**：测试时间长，效率低下
- **目标**：找到方差可接受+时间合理的平衡点
