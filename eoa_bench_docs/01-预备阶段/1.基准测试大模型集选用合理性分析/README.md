# 1.基准测试大模型集选用合理性分析

## 文档概述
本文档分析从MNN支持的160+大模型（含文本、多模态、语音）中选定的10个代表性子集的合理性，确保模型选择的科学性和全面性，为EAO基准测试项目提供大模型测试基础。

## 目录结构
```
1.基准测试大模型集选用合理性分析/
├── 1.基准测试大模型集选用合理性分析.md    # 主分析文档（已完成）
├── 支撑文件/                                # MNN模型列表和初步结论
│   ├── MNN_models_list.csv                 # MNN官方模型清单
│   └── 初步结论.md                         # 初步筛选结论
└── README.md                                # 本文件
```

## 快速开始
1. 阅读主分析文档了解分析方法、维度体系和最终选定列表
2. 查看支撑文件了解MNN实际模型生态和候选范围
3. 参考最终10个模型选定列表进行后续测试规划

## 主要内容
- **分析对象**：从MNN 160+大模型中精选10个代表性模型
- **维度体系**：处理模态、参数规模、厂商生态、应用功能四维度
- **最终选定**：提供10个具体模型的选定列表和覆盖度分析
- **备选方案**：8个模型的灵活选择方案
- **依赖关系**：为2-5号报告提供大模型测试基础
