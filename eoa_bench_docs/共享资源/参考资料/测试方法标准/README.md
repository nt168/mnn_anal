## 📋 使用指南

### 🎯 什么时候需要查阅本目录文档？

**基准测试质量控制时：**
- 需要确定测试结果的可靠性标准
- 需要评估测量方法的精度要求
- 需要制定CV控制目标

**测试方法验证时：**
- 需要验证新测试方法的合理性
- 需要改进测试结果的稳定性
- 需要建立测试质量评估体系

**报告撰写支持时：**
- 需要引用CV的标准依据
- 需要解释测试结果的可信度
- 需要制定改进计划

### 📊 CV标准快速参考

| 应用场景 | 推荐CV标准 | 用途说明 |
|---------|-----------|---------|
| 高精度对比测试 | CV < 10% | 关键性能指标的严格评估 |
| 一般性能评估 | CV < 20% | 常规测试的可接受范围 |
| 参数探索性测试 | CV < 30% | 初步测试的最低要求 |
| 流程稳定性监控 | CV < 15% | 生产过程质量控制 |
