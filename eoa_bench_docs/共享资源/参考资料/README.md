# 参考资料目录指南

## 📁 目录结构

参考资料目录按内容类型组织，分为以下子目录：

### 🔍 [初测结果/](../初测结果/)
**【重要发现】初测阶段数据分析和重大发现**

**核心发现**：llm_bench_prompt输出稳定性远超预期，CV值平均0.05-0.15%，颠覆预设测试方法论基础

**包含文件：**
- `初测结果稳定性分析报告.md` - **【必读】**重大发现和方法论重构建议
- `benchmark_results.db` - 原始SQLite数据库结果
- `README.md` - 初测结果目录使用指南

**影响评估**：🔴 **重大影响** - 需要重新评估整个项目测试策略

**包含文件：**

- `MNN_LLM_Benchmark_YAML配置实用指南.md` - 实用配置指南和最佳实践
- `MNN_LLM_Bench工具总结.md` - MNN LLM Bench工具使用总结
