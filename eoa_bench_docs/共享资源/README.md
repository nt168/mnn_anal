# 共享资源目录

## 一、目录概述
本目录包含EAO基准测试项目的共享资源，包括参考资料和配置文件等。

## 目录结构
```
共享资源/
├── 参考资料/         # 参考资料
│   ├── papers/        # 相关论文
│   ├── specs/         # 技术规范
│   ├── docs/          # 文档资料
│   └── links/         # 外部链接
├── 配置文件/          # 配置文件
│   ├── environments/  # 环境配置
│   ├── models/        # 模型配置
│   ├── hardware/      # 硬件配置
│   └── testing/       # 测试配置
└── README.md         # 本文件
```

## 二、参考资料管理

### 2.1 内容分类
- **技术文档**: MNN框架文档、llama.cpp使用指南
- **学术论文**: 性能建模、机器学习推理优化相关论文
- **规范标准**: 测试标准、数据格式规范、性能指标定义
- **行业报告**: 端侧AI推理发展趋势和技术报告

### 2.2 引用格式
- 使用标准引用格式记录文档信息
- 包含作者、标题、出版日期、访问链接
- 区分内部文档和外部资源
- 标注文档的重要性和适用范围

## 三、配置文件管理

### 3.1 环境配置
- **开发环境**: IDE配置、依赖包列表
- **测试环境**: 硬件配置、软件版本
- **生产环境**: 部署配置、运行参数

### 3.2 模型配置
- **测试模型**: 模型路径、参数设置
- **量化配置**: 量化类型、精度要求
- **优化配置**: 编译选项、优化级别

### 3.3 硬件配置
- **CPU配置**: 核心数、频率设置
- **GPU配置**: 显存、计算单元设置
- **内存配置**: 大小、访问模式设置

### 3.4 测试配置
- **测试参数**: 参数值范围、组合方式
- **测试场景**: 测试用例、执行顺序
- **数据采集**: 指标定义、采集频率
