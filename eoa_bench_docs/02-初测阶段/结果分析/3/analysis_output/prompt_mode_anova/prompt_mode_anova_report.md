## 2. 分析方法
### 2.1 单因素方差分析(One-Way ANOVA)
- 原假设H0: 三种提示词模式(VP0, VP1, PF_FILE)的性能均值相等
- 备择假设H1: 至少有一种模式的性能均值与其他模式不同
- 显著性水平α=0.05

### 2.2 方差齐性检验
- Levene检验: 检验各组方差是否相等
- Bartlett检验: 另一种方差齐性检验方法

### 2.3 事后检验
- Tukey HSD: 当ANOVA显著时，进行两两比较
- 控制多重比较中的第一类错误率
