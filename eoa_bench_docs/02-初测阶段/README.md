# 初测阶段

## 一、阶段概述
初测阶段是在预备阶段基础上进行广泛的参数组合测试，主要目标是确定端侧AI推理优化测试参数的有效取值范围、运行时间优化和关键测试参数筛选，为建立端侧设备LLM推理性能数学模型奠定基础。

## 二、主要任务
1. **使用更多的测试参数组合进行广泛测试**
2. **选定测试参数的取值范围**
3. **选定具有较短运行时间的关键测试参数**

## 五、关键输出
### 5.1 参数范围确定
- 有效参数取值范围列表
- 无效参数排除清单
- 参数敏感度排序

### 5.2 时间优化目标
- 单次测试时间目标值
- 批次测试优化方案
- 总测试时间控制策略

### 5.3 关键参数筛选
- 基于运行时间影响的参数排序
- 基于性能影响的参数权重
- 最终关键参数清单

## 六、测试策略
- 采用分组测试策略，按参数类别组织测试
- 使用正交测试设计减少测试组合数量
- 实施渐进式参数筛选，逐轮优化

## 七、成功标准
- 完成至少80%的目标参数范围测定
- 关键参数筛选收敛，筛选结果稳定
- 测试时间控制在目标范围内
