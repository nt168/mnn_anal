# 模型性能方差分析报告

## 方差分析发现

通过对16个完整数据样本的双因子方差分析（Two-way ANOVA），我们获得了以下重要统计发现：

### 🎯 核心统计结果

| 效应类型 | 平方和 | 自由度 | 均方 | F值 | p值 | 效应量(η²) |
|---------|-------|-------|------|-----|-----|----------|
| **模型主效应** | 69.854 | 1 | 69.854 | 15.345 | **0.002** | 0.413 |
| **环境主效应** | 34.047 | 1 | 34.047 | 7.479 | **0.018** | 0.201 |
| **交互效应** | 10.812 | 1 | 10.812 | 2.375 | 0.149 | 0.064 |
| 误差 | 54.627 | 12 | 4.552 | - | - | - |
| 总计 | 169.340 | 15 | - | - | - | - |

### 📊 关键统计学发现

#### 1. **模型主效应：高度显著 (p = 0.002)**
- **F统计量**: 15.345，远超临界值
- **效应量**: η² = 0.413，说明41.3%的总变异由模型差异解释
- **实际意义**: Qwen vs Hunyuan的性能差异具有统计学意义
- **Hunyuan优势**: 平均速度快4.316 tokens/s (24.630 vs 20.314)

#### 2. **环境主效应：显著 (p = 0.018)**
- **F统计量**: 7.479，达到统计显著水平
- **效应量**: η² = 0.201，20.1%的变异由硬件环境解释
- **实际意义**: 物理vs模拟硬件差异真实存在
- **模拟环境优势**: 平均速度快2.917 tokens/s (23.391 vs 20.474)

#### 3. **交互效应：不显著 (p = 0.149)**
- **F统计量**: 2.375，未达到显著水平
- **效应量**: η² = 0.064，仅6.4%变异由交互作用解释
- **实际意义**: 模型类型与硬件环境之间没有显著交互效应

### 🔍 数据质量影响分析

#### 数据完整性问题
- **总样本**: 20个查询结果
- **可用样本**: 16个（去除512 token截断数据）
- **截断比例**: 20% (4/20)
- **截断模式**: 所有截断都发生在Hunyuan模型的符号和科学查询上

#### 截断对结果的影响
1. **可能低估了Hunyuan的真实性能**：截断阻止了完整输出
2. **增加了方差**：不完整数据增加了误差变异
3. **偏向保守估计**：如果考虑截断因素，实际差异可能更大

### 📈 效应量解释

#### 大效应标准 (Cohen, 1988)
- **大效应**: η² ≥ 0.14
- **中等效应**: 0.06 ≤ η² < 0.14
- **小效应**: η² < 0.06

#### 本研究发现
- **模型效应**: η² = 0.413 → **大效应** (非常重要)
- **环境效应**: η² = 0.201 → **大效应** (重要)
- **交互效应**: η² = 0.064 → **中等效应** (次要)

### 🏆 统计学结论

#### 强证据支持
1. **模型性能差异真实存在** (p < 0.01)
2. **Hunyuan 0.5B 在解码速度上显著优于 Qwen 0.6B**
3. **模拟硬件环境显著优于物理硬件环境**

#### 无证据支持
1. **模型与环境的特殊交互效应**
2. **某个模型在特定环境下有异常表现**

### 📋 模型选择建议（基于统计证据）

#### 选择Hunyuan 0.5B当：
- **性能优先**：统计证实的速度优势
- **高频交互**：累计响应时间节省显著
- **资源充足**：考虑可能的输出长度影响

#### 选择Qwen 0.6B当：
- **输出稳定性优先**：无截断问题
- **资源受限环境**：相对一致的输出长度
- **预测性要求高**：变异性更小(σ=0.287 vs 2.905)

#### 环境建议
- **优先使用模拟硬件环境**：统计证实性能提升
- **但需验证实际业务场景的效果**

### ⚠️ 统计学局限

1. **样本量小**：共16个有效样本，统计功效受限
2. **数据不完整**：20%数据被截断，可能影响估计精度
3. **场景单一**：仅5种查询类型，推广性有限
4. **无重复测量**：无法评估测量稳定性

### 🔬 进一步研究建议

1. **扩大样本规模**：增加查询类型和重复次数
2. **解决截断问题**：调整输出限制以获得完整数据
3. **多指标综合评估**：加入质量、准确性等维度
4. **实际场景验证**：在真实应用场景中验证统计发现

---

## 最终结论

**方差分析提供了强有力的统计证据**：在解码速度方面，Hunyuan 0.5B显著优于Qwen 0.6B（p<0.01，大效应量），模拟硬件环境显著优于物理硬件环境（p<0.05，大效应量）。然而，数据质量问题建议我们保持一定谨慎，需要更大规模和更完整的数据来确认这些发现的稳定性和推广性。
